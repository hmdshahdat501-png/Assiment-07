# 📊 Timeline Tracker App

## 📝 Description
This is a React-based web application where users can track their communication activities like calls, messages, and video interactions. The app allows filtering and searching data easily and provides a visual representation using charts.

---

## ⚙️ Technologies Used
- React.js  
- React Router  
- Context API  
- Tailwind CSS  
- Recharts  

---

## 🚀 Key Features
- Filter activities by Call, Message, and Video  
- Search data by user name  
- Visual statistics using Pie Chart
